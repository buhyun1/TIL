파일 다운로드 받는 방법

방법링크
1. 절대경로 anchor tag의 download 속성 smse_emblem.png 받기
2. 상대경로 anchor tag의 download 속성 univ_logo00.png 받기
3. anchor tag의 download 속성: 파일명 Non-ASCII 파일다운로드.pdf 받기
4. Servlet을 이용한 다운로드 직접 구현 FileDownload.docx 받기
5. Servlet을 이용한 다운로드 직접 구현: 파일명 Non-ASCII 파일다운로드.pdf 받기
6. JSP를 이용한 다운로드 직접 구현 파일다운로드.pdf 받기


'upload' 폴더의 파일 다운받기 (POST 방식)
